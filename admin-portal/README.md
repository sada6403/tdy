# NF Plantation Admin Portal

This is the professional finance-grade back-office system for NF Plantation.

## Tech Stack
- React + Vite
- Lucide React (Icons)
- Recharts (Analytics)
- Axios (API Integration)
- Vanilla CSS (Custom Design System)

## Key Features
- **Dashboard**: High-level operational overview & growth tracking.
- **Verification Pipeline**: KYC and customer registration approvals.
- **Financial Operations**: Deposit & withdrawal management with treasury sync.
- **Customer Registry**: Global user management with status control.
- **Branch Management**: Physical infrastructure configuration.
- **Reports & Audit**: Performance tracking and administrative audit trail.
- **System Settings**: Product (Plan) design and security protocols.

## Getting Started
1. `npm install`
2. `npm run dev`

## Backend Integration
The portal is pre-configured to proxy `/api` requests to `http://localhost:5000`. Ensure the backend server is running.
